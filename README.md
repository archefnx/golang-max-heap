# Go Heap Example

Простой пример реализации **бинарной кучи (heap)** на языке Go.

Репозиторий содержит минимальную реализацию **max-heap** с использованием стандартного пакета Go:

`container/heap`

Цель проекта — показать, как работает структура данных **heap** и как она реализуется через интерфейс `heap.Interface`.

---

# Что такое Heap

**Бинарная куча (Binary Heap)** — это структура данных, основанная на **бинарном дереве**, которая обычно реализуется через **массив (slice)**.

Основное свойство max-heap:

parent >= left_child
parent >= right_child


Это означает, что **корень всегда содержит максимальный элемент**.

---

# Представление в массиве

Heap обычно хранится в массиве:

Index: 0 1 2 3 4 5
Value: [9, 7, 6, 5, 4, 3]


Связь элементов:

parent = (i-1)/2
left = 2i + 1
right = 2i + 24
